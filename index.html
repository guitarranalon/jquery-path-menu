Menú Path usando jQuery

El último lunes vi la implementación de Victor Coulon del menú de Path y me apeteció probar a ver si era capaz de lograr un resultado parecido con jQuery, pero sin invertir demasiado tiempo. La mayor parte de los estilos los tomé de la página de Victor Coulon, así que quiero darle las gracias desde estas líneas.

¿Por qué hice esto?

Como un mero entretenimiento/ejercicio, así que si necesitas algo parecido existen varias alternativas en la red.

Path menu using jQuery
